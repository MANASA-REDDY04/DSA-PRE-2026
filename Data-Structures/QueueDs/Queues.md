# Queue

First In First Out Data Structure

- A Collection designed for holding elements that are prior to processing
- Linear Data structure

---

- Enqueue (Add) from the end, 
```
offer()
```
- Dequeue (Remove) from the start
```
poll()
```

## Uses

1. Keyboard Buffer (letters should appear in sequence we type)
2. Printer Queue (jobs)
3. Breadth-First-Search (BFS)
